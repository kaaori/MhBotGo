Here is a helpful list of commands and their usages!

Normal commands:

  • !mh birthday set "month/day"
    • Sets your birthdate
    •  !mh birthday set 11/4
  • !mh birthday reset
    •  Resets your birthdate

Event runner commands:

  • !mh events add "mm/dd/yyyy HH:MMpm" "Host" "Event Name" "Location"
    • Adds an event for the given day, if it's in the current week it will refresh the schedule
    •  !mh events add "12/4/2019 11:30pm" "MHChan" "My event!" "My computer"
  • !mh events remove "mm/dd/yyyy HH:MMpm" "Host" "Event name" "Location"
    • Removes an event for the given day, if it's in the current week it will refresh the schedule
    •  !mh events remove "12/4/2019 11:30pm" "MHChan" "My event!" "My computer"
